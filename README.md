# 考前复习进度（骗自己学习）
期末考试前，我遇到了一个很现实的问题。不是没有资料，恰恰相反，是资料太多了。

我根本！学！不！完！了！

课件、作业、真题、历年卷、答案、知识点总结，加起来几十个文件。当所有东西都摆在面前的时候，反而不知道该从哪里开始复习。（更不用说什么马原毛概这种了）

于是我为了骗自己学习，做了一个进度条：把整本高数拆成一个个知识点，每完成一个就点亮一个模块，让进度往前推一点。虽然说这个比较简单（且没有什么可比性），但确实有用，看到”6 / 29“和看到“还有23个知识点没学”完全是两种心态，尤其是结合宋浩的高等数学食用效果更佳。其中功能包括：
* 高数B下知识点拆分
* 通关进度记录
* 金币奖励系统
* 本地自动保存
* 一键重开（不是说大家的考试哈）

以及一点考试周特有的求生欲。

后面复习的时候，我把课件、作业、真题和历年卷都丢给了 Codex，它帮我：
* 整理知识点
* 归纳题型
* 总结解题方法
* 制定复习顺序

真正有价值的是这个思路：
当任务大到不知道怎么开始的时候，先别想着努力，
先想办法让自己开始。

最后祝大家考试顺利
